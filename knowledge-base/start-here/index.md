# Start Here

This section is for entry-point material. It should orient new readers without
turning into a catch-all storage area for every foundational doctrine.

Pages placed here should help readers understand:

- what Catholic Veritas is
- how to use the knowledge base
- where different kinds of readers should begin
- the first core questions that unlock the rest of the site

Long-term doctrine pages should eventually live in their proper canonical
domains rather than remaining here permanently.
