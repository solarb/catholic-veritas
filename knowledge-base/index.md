# Knowledge Base Overview

Catholic Veritas is being built as a long-term Catholic apologetics and
theological reference library. This section is the public-facing content layer
of the project.

The governance documents in this repository define how the knowledge base
should grow, how claims should be classified, how sources should be weighed,
and how new pages should fit into the site's long-term architecture.

## Current Focus

The first development phase is establishing the permanent foundation:

- governance
- architecture
- research standards
- content planning
- article planning rules

## Core Domains

The starter structure below mirrors the major domains already defined in the
project governance:

- Start Here
- Scripture
- The Church
- Faith
- Apologetics

These section overviews exist to give the site a durable skeleton before large
numbers of articles are added.
