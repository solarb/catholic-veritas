# Project Context

## Catholic Veritas

This document is the highest-level project reference for Catholic Veritas. It defines the identity, mission, long-term vision, current development stage, technical environment, and decision-making priorities of the project. It is intended to function as a permanent governing reference for all future contributors, whether human or AI.

Every future AI assistant or collaborator working on Catholic Veritas should read this document first before making recommendations, planning architecture, proposing content structures, or modifying project direction.

## Project Overview

Catholic Veritas is a long-term Catholic apologetics and theological knowledge base. It exists to become a durable, trustworthy, and carefully structured reference library that helps readers discover, understand, and defend the Catholic faith.

This project should not be treated as a temporary website, a short-term publishing effort, or a content experiment. It should be developed as a lasting body of work whose articles remain useful, relevant, and intelligible for years into the future.

The project should steadily grow into a coherent knowledge base that integrates theology, apologetics, Scripture, history, and Catholic teaching in a way that is faithful, structured, and maintainable over the long term.

## Mission

The mission of Catholic Veritas is to help people discover, understand, and defend the Catholic faith through Sacred Scripture, Sacred Tradition, reason, history, and the continuous teaching authority of the Catholic Church.

Its purpose is to present Catholic truth faithfully, explain it clearly, defend it responsibly, and preserve it in a form that remains accessible and useful to future readers.

## Long-Term Vision

Catholic Veritas should become a trusted long-term Catholic reference website and knowledge base that serves multiple purposes at once.

Over time, it should develop into:

- a searchable apologetics database
- a theological reference library
- an evangelisation resource
- a research companion
- a structured study resource
- a trustworthy Catholic reference website for long-term use

The long-term vision is not merely to publish many pages, but to build a coherent and enduring body of Catholic knowledge that readers can return to repeatedly for study, clarification, and defense of the faith.

## Target Audience

Catholic Veritas is intended to serve readers with different backgrounds, beliefs, and levels of knowledge.

Its audience includes:

- Catholics seeking deeper understanding of their faith
- Protestants exploring Catholic teaching and objections
- Orthodox Christians examining points of agreement and difference
- Muslims interested in Christian and Catholic claims
- atheists and skeptics evaluating the truth claims of Christianity and the Church
- curious readers seeking clear and well-grounded answers
- RCIA / OCIA candidates learning the foundations of the faith
- students studying theology, Scripture, or Church history
- teachers, catechists, and writers who need dependable reference material

Because the audience is broad, all content should be written with the assumption that readers will possess varying levels of biblical, theological, philosophical, and historical knowledge.

## Current Development Stage

Catholic Veritas is currently in the Foundation Phase.

At this stage, the project should prioritize the long-term integrity of the knowledge base rather than surface-level polish or short-term expansion.

The current priorities are:

1. Build accurate content.
2. Build consistent architecture.
3. Establish internal linking.
4. Preserve long-term maintainability.

Visual polish should not be prioritized over content quality, doctrinal reliability, architectural consistency, or long-term usefulness.

Free Plan

All recommendations for Catholic Veritas should work within the current platform and plan unless Peter explicitly decides to migrate or upgrade.

Contributors should assume that the current environment does **not** provide access to:

- plugins
- custom themes
- PHP
- FTP
- server configuration
- databases
- custom JavaScript

Recommendations should therefore prefer native WordPress Block Editor functionality whenever possible.

If a proposed idea depends on functionality that is unavailable on the current platform, that idea should be presented only as an optional future enhancement rather than as the default recommendation.

## Guiding Principles

The development of Catholic Veritas should be governed by the following principles:

- Truth before popularity.
- Accuracy before speed.
- Scripture before opinion.
- Primary sources before secondary sources.
- Clarity before complexity.
- Charity before polemics.
- Long-term maintainability before short-term convenience.
- Faithfulness to the Catholic Church above personal interpretation or novelty.

These principles should shape content decisions, structural decisions, editorial standards, and future recommendations.

## Success Criteria

Success for Catholic Veritas is **not** measured primarily by:

- number of articles
- number of visitors
- visual effects
- plugins
- advanced technology

Success **is** measured by:

- theological accuracy
- historical reliability
- biblical faithfulness
- organisational consistency
- readability
- usefulness
- trustworthiness
- long-term value

The project should aim to become a site that readers trust because its content is careful, clear, faithful, and durable.

## Future Growth

This document should evolve over time as Catholic Veritas matures.

If the project changes platform, upgrades its technical environment, expands its scope significantly, or enters a new stage of development, `PROJECT_CONTEXT.md` should be updated before architectural, editorial, or technical recommendations are changed to match the new reality.

This document should remain the first reference point for future planning so that the identity and direction of Catholic Veritas remain stable even as the project grows.
